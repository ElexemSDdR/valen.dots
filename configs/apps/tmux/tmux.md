# Configuración de Tmux

## Plugins
Los plugins los instalamos con TPM

**Para instalar TPM**
```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

Luego para instalar los plugins dentro de tmux tenemos que hacer `Ctrl + Shift + i` y aparecerá una pequeña pestaña indicando que se está instalando


## Archivos de Configuración
Los plugins y las configuraciones de tmux (*.tmux.conf* y *.tmux/*) las pondremos directamente en la carpeta home (*~/*)
